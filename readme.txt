== Installation ==

1. Upload `aspexi-facebook-side-box` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin (Appearance -> Facebook Like Box Sidebox)

[Premium version only](http://aspexi.com/downloads/aspexi-facebook-like-box-sidebox-hd/?src=plugin_page):

* Remove plugin's free version before install premium version
* Put license code from confirmation email into plugin configuration and click Activate License

== Frequently Asked Questions ==

= I see blank box, why is that? =

Double check if your Facebook page is published and available to all users.

= Where do I find settings page in Admin? =

Simply go to Appearance -> Facebook Like Box Sidebox

= Can I use my custom Facebook button image? =

Yes. Simply create your file and copy it to your current theme (or child) directory:

* **required file URL**: TEMPLATE_DIR/plugins/aspexi-facebook-side-box/images/aspexi-fb-custom.png

* **required image width**: 48px
